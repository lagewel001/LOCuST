# INSTRUCTIE:
Genereer een SQL-query die de vraag beantwoordt. Genereer ALLEEN de SQL-query, en geen enkele andere tekens.
Als input krijg je de vraag en voorbeeldtabellen met de bijbehorende measures en dimensions. Indien in een
retrieval-stap scores zijn berekend van tabellen of measures/dimensions worden die ook meegegeven. Voor elke
measure wordt ook de eenheid meegegeven. De dimensies voor 'Perioden' en 'RegioS' worden hier niet getoond
omwille van de beknoptheid, ongeacht of ze onderdeel zijn van een tabel.

Gebruik de volgende instructies voor het schrijven van SQL query's:
- Tabellen kunnen naar gerefereerd worden door middel van `FROM "data/nl/odata3/<table-id>.parquet"`
- De SQL antwoorden moeten resulteren in een draaitabel over de measures. Om dit grofweg te bereiken,
  kunnen query's als volgt gestructureerd worden:
```
SELECT
    *
FROM (
  SELECT <select cols>
  FROM "data/nl/odata3/<table id>.parquet"
  UNPIVOT(Value FOR Measure IN (<measures>))
  WHERE <where clause>
)
PIVOT(<pivot function>(Value) FOR
  <dimensions>
)
```
